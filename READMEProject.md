# Music Playlist using Circular Linked List (C)

This project implements a simple **music playlist system** using a **circular linked list** in C.  
Each song is stored as a node, and the playlist loops infinitely.

## Features Implemented

### ✔ Add new song  
Insert a new song at the end of the playlist.

### ✔ Delete song  
Remove a song by title.

### ✔ Move to next / previous  
Navigate through the playlist just like a music player.

### ✔ Jump to head / tail  
Directly go to the first or last song in the playlist.

### ✔ Display playlist  
Shows all songs and marks the currently selected song.

## How to Compile

Use GCC:

```bash
gcc -o playlist main.c

How to Run:
./playlist

Menu Options

1. Add new song
2. Delete song
3. Move to next song
4. Move to previous song
5. Jump to first song
6. Jump to last song
7. Display playlist
8. Exit
